# Automatic Number Plate Recognition (ANPR) System

**Year 1 Semester 1 Project**  
Authors: **Leong Yao Cheng, Kwah Wen Yao, Tan Pei Yu, Zhang Yun, Liu Pinyi**

## Project Description
This project implements an **Automatic Number Plate Recognition (ANPR)** system using classical image processing and machine learning techniques.
The system processes an input vehicle image, detects the license plate region, extracts the characters, and recognizes the final plate number.
